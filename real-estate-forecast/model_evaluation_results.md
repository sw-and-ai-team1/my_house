### 5.3 실험 3: 특성공학 (Feature Engineering)

#### 5.3.1 추가된 특성

| 특성명             | 설명             | 계산 방법                        |
| ------------------ | ---------------- | -------------------------------- |
| **price_per_area** | 면적당 가격      | dealAmount_now / excluUseAr      |
| **age_group**      | 건물 연식 구간화 | 0-5년, 5-10년, 10-20년, 20년+    |
| **floor_group**    | 층 구간화        | 저층(1-5), 중층(6-15), 고층(16+) |
| **season**         | 계절성           | 거래월 → 계절 변환               |

#### 5.3.2 특성공학 세부사항

##### 연식 구간화 (age_group)

```python
age_group = pd.cut(age_at_deal,
                   bins=[0, 5, 10, 20, float('inf')],
                   labels=[0, 1, 2, 3])
```

- 0: 신축 (0-5년)
- 1: 준신축 (5-10년)
- 2: 중간 (10-20년)
- 3: 노후 (20년 이상)

##### 층 구간화 (floor_group)

```python
floor_group = pd.cut(floor,
                     bins=[0, 5, 15, float('inf')],
                     labels=[0, 1, 2])
```

- 0: 저층 (1-5층)
- 1: 중층 (6-15층)
- 2: 고층 (16층 이상)

##### 계절성 (season)

```python
season = dealMonth.apply(lambda x:
    0 if x in [12, 1, 2] else    # 겨울
    1 if x in [3, 4, 5] else     # 봄
    2 if x in [6, 7, 8] else     # 여름
    3)                           # 가을
```

#### 5.3.3 모델링 및 평가 결과

| 실험 구분         | MAE (만원)   | RMSE (만원)  | R²          | 사용 특성 수 |
| ----------------- | ------------ | ------------ | ----------- | ------------ |
| **기본 특성**     | 3,164.69     | 6,293.26     | 0.9711      | 6개          |
| **특성공학 적용** | **3,155.28** | **6,233.41** | **0.9717**  | 10개         |
| **개선도**        | **+9.41**    | **+59.85**   | **+0.0006** | +4개         |

#### 5.3.4 결과 분석

- **성능 향상**: MAE 9.41만원 개선 (0.3% 향상)
- **안정성 증가**: RMSE 59.85만원 개선으로 예측 안정성 향상
- **설명력 증가**: R² 0.0006 포인트 상승
- **미미하지만 일관된 개선**: 모든 지표에서 소폭이지만 지속적인 성능 향상

- **실용성**: 부동산 투자 의사결정에 충분한 예측 정확도
